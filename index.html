Visualización de secciones con doble modo

Usa el siguiente control para alternar entre ver todas las secciones de forma continua o navegar de a una sección por vez.

Introducción

Esta demostración muestra cómo alternar entre ver todo el contenido en una única vista continua o concentrarse en una sección específica para evitar distracciones.

El modo continuo resulta útil para hacer una lectura rápida. El modo de una sola sección facilita la lectura concentrada, especialmente al utilizar lectores de pantalla o dispositivos móviles.

Estructura

Componentes principales

Cada sección contiene su propio encabezado y puede incluir artículos u otras unidades semánticas, lo que permite mantener una jerarquía clara y accesible.

Accesibilidad

Los elementos interactivos son accesibles por teclado y cuentan con estados de foco visibles. Se utiliza un texto de estado con aria-live para anunciar los cambios.

Proyección

El patrón puede adaptarse para tutoriales, presentaciones o documentación técnica que necesite alternar entre una vista resumida y otra paso a paso.

  • Permite controlar la carga cognitiva.
  • Favorece la navegación guiada.
  • Se integra con lectores de pantalla mediante mensajes en vivo.
